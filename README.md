Descripción del Proyecto
Este proyecto es una aplicación de consola desarrollada en Python que implementa un sistema para gestionar una biblioteca digital. El código aplica conceptos de Programación Orientada a Objetos (POO) y gestiona colecciones de datos complejas para asegurar la eficiencia del sistema.

Estructuras de Datos Utilizadas
Para cumplir con los requerimientos de la asignatura, se utilizaron las siguientes colecciones:

Diccionarios: Se utilizan para el catálogo de libros, donde el ISBN funciona como clave única para búsquedas rápidas.

Tuplas: Almacenan la información inmutable de cada libro, como el Título y el Autor.

Conjuntos (Sets): Gestionan los IDs de los usuarios para garantizar que cada identificador sea único en el sistema.

Listas: Registran los libros que cada usuario tiene bajo préstamo actualmente.

Funcionalidades Principales
Gestión de Libros: Permite añadir, eliminar y buscar libros en el catálogo por diferentes criterios.

Gestión de Usuarios: Registro y baja de usuarios con identificadores únicos.

Préstamos y Devoluciones: Sistema para prestar libros y registrar su retorno, actualizando automáticamente el estado del catálogo.
